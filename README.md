# 🧮 SCT_WD_2-Alternative: Modern Calculator

A sleek, high-performance **Web Calculator** designed with a focus on user experience and minimalist aesthetics. This version features an alternative layout with enhanced precision and a "Dark Mode" inspired interface.

## 🚀 Key Features

* **Precision Arithmetic:** Handles basic and complex operations (Addition, Subtraction, Multiplication, Division).
* **Alternative Design:** Features a compact, modern button grid with high-contrast color coding for operators.
* **Clear & Delete:** Includes specialized functionality for both single-character deletion and full memory clearing.
* **Responsive Layout:** Fully adaptive design that works perfectly on mobile phones, tablets, and desktops.
* **Hover & Active States:** Interactive visual feedback for every button click.

---

## 🛠️ Technical Stack

* **HTML5:** Semantic markup for a clean structure.
* **CSS3:** Advanced styling using **Flexbox** and **CSS Grid**, featuring custom variables for easy theme management.
* **JavaScript (ES6):** Logic-driven event listeners and a sanitized calculation engine.

---

## 📸 Project Goals

> [!IMPORTANT]
> This project focuses on **Input Validation**—ensuring that users cannot enter multiple operators sequentially and maintaining a clean display output throughout the calculation process.
